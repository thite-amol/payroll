How to Fillup Employee Information

Back to top

For Fill up the form, do the following:

  • First, select the form of Add Employee by clicking on the add button which is indicated by addition logo and it given first position of toolbar.
  • The form is opened in front of you then fillup the appropriate information in the form no any one field is empty because empty field is not considered in the information there is exception is shown by software field name is empty fill up that field.Enter full name in the name field and enter the numbers in field where number is requiered otherwise exception is occured
  • After filling the information press the Save button for save the information
  • Select the cancle option if cancle the form saving
  • Select the Modify option if modify the form Employee information
  • On the form tooltips are given for user information
